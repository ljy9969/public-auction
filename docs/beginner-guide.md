# 공매 입문자용 소액 투자 가이드

> 참고 도서: *실전 부동산 경매 (수익 실현편)* — 유근용, 정민우
> 본 문서는 책 내용을 우리 공매(온비드) 환경에 맞게 정리한 발췌·요약본입니다.
> 책 본문 전체를 옮기지 않고 의사결정에 직접 쓸 수 있는 룰만 추렸습니다.

## TL;DR — 입문자에게 가장 잘 맞는 4가지 매물 유형

| 유형 | 평균 투자금 | 난이도 | 핵심 가치 |
|---|---|---|---|
| **① 토지 지분** | 300~3,000만원 | ★★☆ | 공유자 매도(가액배상) 협상으로 1~1.5년 내 회수. 경쟁 적고, 끊임없이 나옴 |
| **② 도로/맹지 인접 부속토지** | 300~1,000만원 | ★★☆ | 옆 땅 주인에게 꼭 필요 → 강제 매수 압력. 책 사례 "341만원 → 200%+ 수익" |
| **③ 소형 빌라·오피스텔(거주용)** | 1~3억 | ★★☆ | 실거주 + 임대 가능. 명도·권리분석 학습 좋음 |
| **④ 농지 지분** | 200~2,000만원 | ★★★ | 농지취득자격증명(농취증) 필수. 농업진흥/보호구역은 피할 것 |

## 책에서 강조한 입문자 원칙

1. **"경험 1순위, 돈 2순위"** — 첫 1~2건은 수익 욕심 버리고 절차를 익혀라
2. **권리분석 핵심은 단 하나**: 말소기준권리(근저당·가압류 80%)만 찾기
3. **비고란**: 매각물건명세서에서 "분묘기지권", "지분 매각임", "맹지" 표기를 가장 먼저 확인
4. **입찰 전 시세**: KB부동산 + 국토부 실거래가 + 주변 매물로 3중 검증
5. **입찰 보증금**: 최저가의 10% — 100원이라도 부족하면 무효 (수표 한 장 권장)

## 유형별 상세

### ① 토지 지분

**왜 입문자에게 좋은가**:
- 항상 새 물건이 나옴 (상속·이혼·공동투자 정리)
- 유찰 多 → 가격 저렴
- 경쟁자 少 (특수물건 알레르기)
- **정형화된 매뉴얼**: 낙찰 → 가처분 → 공유물분할청구소송 → 협상

**처분 5가지 경로**:
1. 내 지분을 공유자에게 매도 (책 권장 — 70~80%는 이 단계에서 정리)
2. 공유자 지분 매입해서 통합 매도
3. 통합 후 부동산 사무소 매도
4. 인접 토지 주인에게 매도
5. 형식적 경매로 분할 매각

**주의**: 1금융권 대출 불가, 2금융권도 20~30%만 가능

### ② 도로/맹지 인접 부속토지

**책 인용 사례 (도로 章)**:
> "공장 입구를 막고 있는 토지를 낙찰. 가격 저렴, 지분, 누군가에게 꼭 필요한 위치 — 이 3가지가 핵심. 341만 원 투자 → 200%+ 수익."

**선별 기준**:
- 모양·크기 무관
- "누군가에게 필요한가?" 가 전부
- 공장·주택 입구를 막고 있는 토지 = 강제 매수 압력↑

**검색 키워드**: "도로", "맹지", "지분 토지"

### ③ 소형 빌라·오피스텔

**우리 앱이 이미 잘 잡는 유형**. 책의 추가 팁:
- 초등학교 인근 → 빌라 수요 안정
- 단독주택은 도로 접면 여부 필수 확인 (맹지 = 신축·매도 어려움)
- 다가구주택은 토지·건물 공시가 + 인근 평당 시세로 환산

### ④ 농지 지분

**책 인용 (12회 유찰된 맹지 지분 사례)**:
- 농지취득자격증명(농취증) 필요 (정부24 신청 → 1~2일)
- **농업진흥/보호구역은 절대 입찰 X** (사용 제한 강함)
- 12회 유찰 = 감정가 대비 29% → 도로 접면 여부에 따라 가치 폭주

## 우리 앱 적용 — 어떻게 매물을 가져올지

| 책의 권장 | 우리 시스템 매핑 |
|---|---|
| 토지 지분 | `srchPrptType=0001` + 카테고리 "토지"/"답"/"전" + shrYn=Y |
| 도로 부속토지 | `srchPrptType=0001` + 카테고리 "도로" 포함 |
| 소형 빌라/오피스텔 | 기존 0007/0005 (이미 수집 중) |
| 농지 지분 | `srchPrptType=0001` + 카테고리 "전"/"답"/"과수원" + shrYn=Y |

### 이번 작업으로 추가/변경된 것

1. **criteria.yaml**: `srch_prpt_types`에 "0001" (토지) 추가
2. **quality.py**: 토지/도로는 건물면적·엘리베이터 체크 스킵
3. **목록 페이지 상단 탭**:
   - **주거** — 주거용건물 (실거주)
   - **용도복합·오피스텔** — 오피스텔 (실거주·임대)
   - **주거 지분** — 주거용건물 + 지분 매물 (투자)
   - **도로** — 도로·토지 지분 (입문자용 소액 투자)

> ⚠️ **현재 우리 지역(송파/강남 + 선릉 3km) 한정**으로 검색합니다. 책의 도로/농지 케이스는 대부분 수도권 외곽이라 우리 결과에는 거의 안 잡힐 수 있습니다. 도로/농지 본격 투자하려면 별도 지역 확장 작업 필요.

## 위험 시그널 (책 + 우리 데이터 결합)

| 신호 | 대응 |
|---|---|
| "대항력 있는 임차인" + 전액 배당 X | 인수 금액 발생 가능 → 입찰가에 반영 or 회피 |
| "분묘기지권 성립 여부 불분명" | 토지 가치 하락 → 보수적 입찰 |
| "지분 매각임" | 가액배상 가능성 검토 후 입찰 (책 권장 경로) |

| 맹지 | 단독 사용 어려움 → 인접 토지 매입 가능성 확인 |
| 유찰 5회 초과 | 권리/물리적 하자 가능성 → 현장답사 필수 |
