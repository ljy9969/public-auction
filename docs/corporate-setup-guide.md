## 0. TL;DR — 한눈에

세제 혜택은 **2단(지방세 + 국세)**으로 나눠서 봐야 함.

| 혜택 | 적용 조건 | 효과 |
|---|---|---|
| **등록면허세 / 취득세 중과 면제** (지방세) | 수도권 *외* 또는 수도권 *산업단지* 안 | 3배 중과 회피 |
| **법인세 50% 감면 5년** (국세, 조특법 §6) | **수도권 *외*에서 창업** | 5년간 법인세 절반 |

- **가산·구로디지털단지 = 국가산업단지** → 등록면허세 면제 ✅ (단, 법인세 감면은 X — 수도권 안이라)
- **판교는 갈라 봐야 함**: 1판교(분당 삼평동)는 일반 업무지구라 **면제 불가**, 제2·3판교(수정구 시흥·금토동)만 도시첨단산업단지로 면제 가능 [Gemini]
- **수도권 외 (청주 등)** → 등록면허세 면제 + **법인세 50% 감면까지 동시** [Gemini 추가]
- 검증은 **토지이음 → 팩토리온/KICOX → 관할 구청 세무과** 3단계 + **주소 일치(도로명 vs 지번) 4단계**까지.
- 비상주 사무실은 **사업자등록 가능 여부·전대 동의서·실사 대응·우편물 관리** 4가지가 핵심.
- **수도권 외에 비상주 잡고 운영자는 수도권 거주** = 세무서 위장 창업 의심 ↑ → 실사 대응 옵션 필수.

## 2.5 ★ 법인 2개 분리 전략 (업종별 세제 혜택 차이) — **운영자 확정안**

### 왜 분리하나

조특법 §6 **법인세 5년 50% 감면**의 "창업 중소기업" 범위는 업종별로 한정:

| 업종 분류 | 조특법 §6 50% 감면 대상 |
|---|---|
| 소프트웨어 개발·공급 / 정보통신업 / 플랫폼 서비스 | ✅ 적용 |
| 부동산 매매업 / 임대업 / 개발 및 공급업 | ❌ **원칙적으로 제외** (지식기반산업/제조업 등이 대상) |

→ 부동산 활동을 같은 법인 안에 끼우면 **감면 대상 업종에서 떨어질 수 있음**(주업종 판단·매출 비중 기준).
→ **법인을 둘로 분리**하면 소프트웨어 법인은 50% 감면을 안전하게 받고, 부동산 법인은 별도 세제 전략(법인세 일반세율, 임대소득 관리) 적용 가능.

> ⚠️ 정확한 업종별 §6 적용 여부는 **세무사 확인 필수**. 본 문서는 일반론 + 사용자 확정 방향.

## 3. 주소지 선정 — 가산 · 구로 · 판교 · 수도권 외 비교

### 핵심 결론 (수도권 안 vs 밖)

| 지역 | 산업단지 / 권역 | 등록면허세 중과 | 법인세 50% 감면 | 추천도 |
|---|---|---|---|---|
| **가산디지털단지** (서울 금천구) | 서울디지털 **국가**산업단지 | ✅ 면제 | ❌ (수도권 안) | ⭐⭐⭐⭐⭐ |
| **구로디지털단지** (서울 구로구) | 서울디지털 **국가**산업단지 | ✅ 면제 | ❌ (수도권 안) | ⭐⭐⭐⭐⭐ |
| **제1판교테크노밸리** (분당구 삼평동) | 일반 업무지구 | ❌ 중과 적용 | ❌ (수도권 안) | ⚠️ 주의 |
| **제2·3판교** (수정구 시흥·금토동) | 도시첨단산업단지 | ✅ 가능 | ❌ (수도권 안) | ⭐⭐⭐ (매물 적음) |
| **수도권 외** (청주, 천안 등 충청·지방) | (해당 없음) | ✅ **자동 면제** | ✅ **5년 50% 감면** | ⭐⭐⭐⭐⭐ (절세 최강) |

> ⚠️ **시장에 "판교 비상주"로 유통되는 매물의 대부분은 제1판교(삼평동) 주소** — 등록면허세 3배·법인 자본금의 1.2%가 그대로 부과될 수 있음. [Gemini]

> 💰 **법인세 50% 감면**은 *국세*, 등록면허세는 *지방세*. 수도권 안 산업단지(가산/구로)는 지방세만 면제되고 법인세 감면은 못 받음. 절세 효과 극대화는 **수도권 외 + 비상주** 조합. [Gemini 추가]

### Gemelos Estates (법인 B) — 주소 후보 비교

## 7. 빠른 체크리스트

### 공통
- [ ] 상호 중복 확인 (등기소)
- [ ] 정관 목적사업에 부동산 + 소프트웨어 모두 포함
- [ ] 운영업체 자가/전대 동의 여부 확인
- [ ] 업종 등록 가능성 사전 확인
- [ ] 계약서에 위 ①~⑦ 특약 모두 명시
- [ ] 우편물·실사 대응 옵션 확인
- [ ] 법인 설립등기 → 사업자등록 → 통장 개설 → 4대보험 가입

### 수도권 산업단지(가산/구로/제2·3판교) 선택 시
- [ ] 토지이음에서 "과밀억제권역" + "국가산업단지" 동시 확인
- [ ] 팩토리온에서 단지 경계 안인지 번지 대조
- [ ] 관할 구청 세무과 유선 확인 + 답변 메모

### 수도권 외(청주 등) 선택 시 [Gemini 추가]
- [ ] 토지이용계획확인서에 "과밀억제권역" 문구 *없음* 확인
- [ ] **계약서 도로명 주소 ↔ 확인서 지번**이 같은 건물인지 대조 (네이버 지도 + 건축물대장)
- [ ] 관할 시·구 세무과에 "조특법 §6 50% 감면 대상 맞나" 확답
- [ ] 운영자 거주지(수도권)와 사무실(지방) 거리 → 위장 창업 대비 실사 지원·우편물 즉시 스캔 옵션 계약 명시

### 2법인 분리 운영 시 (Unbound Apex Systems + Gemelos Estates)
- [ ] **세무사와 §6 적용 업종 사전 검토** — Unbound Apex Systems 주업종이 §6 대상 명확한지
- [ ] 두 법인 정관 목적사업 **명확히 분리** (소프트웨어 vs 부동산 — 겹치지 않게)
- [ ] Gemelos Estates는 코워크시티 청주 상당점 *외* 별도 비상주 확보 (1주소 다법인 불가)
- [ ] 두 법인 간 거래는 시가 기준 (특수관계인 부당행위계산 부인 대비)
- [ ] 통장·4대보험·세무 신고는 법인별 *완전 별도*
