## 0. TL;DR

세제 혜택은 **2단(지방세 + 국세)**으로 봐야 함.

| 혜택 | 적용 조건 | 효과 |
|---|---|---|
| **등록면허세·취득세 중과 면제** (지방세) | 수도권 *외* 또는 수도권 *산업단지* 안 | 3배 중과 회피 |
| **법인세 50% 감면 5년** (국세, 조특법 §6) | **수도권 *외*에서 SW·정보통신 등 §6 대상 업종 창업** | 5년간 법인세 절반 |

**확정 운영안:** 2법인 분리 — 그룹 **Gemelos Holdings** 아래 개발 **Unbound Apex Systems**(청주, §6 감면) + 부동산 **Gemelos Estates**(별도 비상주, §6 비대상). MMA 앱 **OctaLink**는 Unbound Apex Systems 산하.

## 2. ★ 법인 2개 분리 전략 — **확정안**

### 왜 분리하나

조특법 §6 50% 감면 대상은 **소프트웨어/정보통신/플랫폼 등 지식기반 업종** 한정. 부동산 매매·임대업은 원칙적으로 §6 비대상. 한 법인에 부동산을 끼우면 주업종 판단·매출 비중에서 감면 박탈 위험 → **분리해서 SW 법인만 §6 50% 감면 안전 확보**.

> ⚠️ 정확한 업종별 §6 적용 여부는 **세무사 확인 필수**.

### 명명 체계

| 역할 | 명칭 |
|---|---|
| 그룹 | **Gemelos Holdings** |
| 개발 법인 (A) | **Unbound Apex Systems** |
| 부동산 법인 (B) | **Gemelos Estates** |
| MMA 앱 | **OctaLink** (Unbound Apex Systems 산하) |

### 2개 법인 비교

| 항목 | 법인 A — Unbound Apex Systems | 법인 B — Gemelos Estates |
|---|---|---|
| 주된 업종 | SW 개발·플랫폼·앱(OctaLink) | 부동산 매매·임대·경공매·개발 |
| 본점 | **청주 상당구 무농정로 6** (코워크시티 청주 상당점) | **별도 비상주** (1주소 다법인 불가) |
| 등록면허세 | ✅ 자동 면제 | 주소에 따라 |
| §6 50% 감면 | ✅ 5년 적용 | ❌ 비대상 |
| 자본금 | 1,000만 원↑ | 1,000만 원↑ (투자 규모에 따라 증액) |

### 분리 운영 시 유의

- 두 법인 간 거래는 **시가 기준** (특수관계인 부당행위계산 부인)
- 동일 대표자 OK (1인 다법인)
- 통장·4대보험·세무 신고는 법인별 *완전 별도*

### 정관 목적사업 — 두 법인 분리 리스트

> 💡 정관에 그대로 옮길 수 있음. 등기 전 세무사 검토 1회 필수. 두 리스트는 **겹치지 않도록** 설계.

7. 디지털 콘텐츠 개발 및 공급업

## 3. 주소지 선정 비교

| 지역 | 등록면허세 | §6 50% 감면 | 비고 |
|---|---|---|---|
| **수도권 외** (청주·천안·대전 등) | ✅ 자동 면제 | ✅ 5년 적용 | **절세 최강** |
| **가산·구로디지털단지** (국가산업단지) | ✅ 면제 | ❌ (수도권 안) | 서울 접근성 |
| **제2·3판교** (도시첨단산업단지) | ✅ 가능 | ❌ | 매물 매우 적음 |
| **제1판교** (분당 삼평동, 일반 업무지구) | ❌ 중과 적용 | ❌ | ⚠️ 시장에 "판교 비상주"로 유통되는 대부분 |
| **수도권 안 비과밀** (강화·옹진·동두천·파주·포천) | ✅ 면제 | ❌ (여전히 *수도권 안*) | [Gemini 보정] |

> ⚠️ "수도권 비과밀 ≠ 수도권 외". 수도권정비계획법상 수도권은 **서울·경기·인천 전체** — 인천 강화·옹진, 경기 포천도 다 수도권 안.

### 조특법 §6 적용 조건 (2026년 창업)

- **수도권 외**에서 **§6 대상 업종** (SW·정보통신·지식기반 등) 으로 창업
- 인구감소지역 여부 무관 — "수도권 외" + "대상 업종" 단일 요건
- 청년 창업 아니어도 적용

### 5.2 운영사 Q&A — 코워크시티 청주 상당점 (2026-06)

| # | 항목 | 답변 요약 |
|---|---|---|
| 1 | **전대 동의** | 건물주와 적법한 전대 동의 완료. 계약 시 전대차계약서 + 전대동의서 제공. 요청 시 건물주 **인감 날인본** 별도 발급 가능 |

| 3 | **임대 만료 대책** | 만료 시 주소 사용 자동 종료 ❌ (종료 원하면 사업자등록증 주소 변경 or 폐업 증명서 제출). 임대인 사정으로 주소 제공 불가 시 → **다른 지점 이전 안내 + 잔여 기간 일할 환불** |
| 4 | **운영사 사업자등록** | 업태: **부동산업** / 업종: **비거주용 건물 임대업 (공유오피스 전대)** |
| 5 | **건물 / 근저당** | 교회 건물에 위치 + **신협 근저당 1억 6,500만 원 설정**. 다른 계약자들도 사업자등록 진행 중이나, **최종 승인은 세무서 담당 주무관 판단** — 사전 보장 불가 ⚠️ |

| 8 | **지역 창업 혜택** | 코워크시티 마이페이지 → **정부지원금(BETA)** → "창업중소기업" / "청주" 키워드로 K-Startup 공고 조회 |

> ⚠️ **Q5 근저당 리스크**: 신협 1.65억 근저당이 잡혀 있어 건물 압류·경매 시 사업장 이용권에 영향이 갈 수 있음. 가격 메리트(월 2만 원) 대신 감수해야 하는 부분. §5.3 특약 ⑥ "운영사 폐업 시 30일 내 대체 주소 제공/환불"이 안전망 역할.

> ⚠️ **Q2 + Q7 결합**: 직원 비상주 + 실사 사전 예약 + 소명 자료 대표자 직접 준비 → 청주 비상주는 **실사 일정·자료를 본인이 적극 관리**해야 함. 우편물 즉시 스캔 옵션은 활용 가능.

### 5.3 계약 필수 체크 4가지

1. **사업자등록/법인등기 가능 여부**
2. **건물주 전대 동의서** (서면) — 불법 재임대면 사업자등록 직권취소 위험
3. **실사 대응** — 회의실/공간 제공 + 매니저 응대 (코워크시티 청주는 *3일 전 예약 필수*)
4. **우편물 관리** — 도착 알림·스캔·정기 배송. 관공서 등기 반송 시 사업자등록 직권취소 위험

### 5.5 수도권 외 비상주 — 위장 창업 의심 리스크

운영자 거주지(수도권)와 본점(청주)이 멀면 세무서가 **페이퍼 컴퍼니** 의심:
- 사업자등록 시점 현장 실사
- 등록 후 6개월~1년 정기 점검 / 해명 자료 요구
- 인정 안 되면 §6 감면 추징

**대응 옵션:**
- 운영사의 실사 자리 제공·매니저 응대 옵션 확인 ✅
- 우편물 즉시 스캔·전송 옵션 ✅
- 회의실 없는 코워크시티 청주는 외부 대관/카페 영수증·미팅 기록으로 활동 흔적
- 전기·수도 사용량 대신 회의·계약 서류 보유
